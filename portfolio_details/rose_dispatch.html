Project information

  • Name: RoseDispatch
  • Category: Logistics, Data management
  • Project year: 2021
  • Mode: Fullstack
  • Technologies: Reactjs, Laravel, Mysql, Google sheet api, G-Maps api, G-Drive api

Some details about the project

A web app that use the Google Sheets API to automatically populate a Google Sheet with data submitted through the form. This allows our client to quickly and easily collect and organize data in a spreadsheet format, without the need for manual data entry. The web form is designed with a clean and intuitive interface, making it simple for users to input data and submit it to the corresponding Google Sheet. Behind the scenes, the Google Sheets API handles the integration between the web form and the corresponding Google Sheet, automatically updating the sheet with any new data submitted through the form. This automation eliminates the need for manual data entry, saving users time and reducing the risk of errors.
